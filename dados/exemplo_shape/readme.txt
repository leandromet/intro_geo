Arquivo exemplo armazenamento disco, elementos:
shp - dados vetoriais
dbf - Dados tabulares, pode ser aberto com openoffice mas se editar pode perder a relação com o shp
prj - dados de projeção (preferencia usar)
shx - índice da geometria se houver (ajuda a processar)
qix - índice espacial gerado pelo qgis, melhor e utiliza btree
